# 루나문의 Processing 작업실🖋

저와 함께 프로세싱 작품을 만드실 분을 구합니다😃

## 👋 프로젝트 소개

이 프로젝트는 **Processing(프로세싱)** 을 사용해서
함께 창의적인 코딩 작품을 만들어보는 프로젝트에요.

**Processing**을 처음 해보는 사람도 괜찮아요!
같이 배우면서 재미있게 만들어요😊

## ❔ 프로세싱이란

### Processing(프로세싱)
미디어 아트 및 교육용 언어 중 하나입니다
원래는 Java를 베이스로 사용하지만
저는 Javascript 라이브러리를 사용해요!

### P5.JS
- [레퍼런스](https://p5js.org/ko/reference/)
- [예제](https://p5js.org/ko/examples/)

## 🎯 프로젝트 목표

- **Processing**으로 재미있는 시각 작품 만들기
- 코딩과 예술을 함께 경험하기
- 협업을 통해 서로 배우기
- 완성된 작품을 공유하거나 전시하기

## 🧩 이런 친구를 찾고 있어요

- 프로세싱이나 코딩에 관심 있는 학생
- 그림, 디자인, 애니메이션, 인터랙션에 관심 있는 사람
- 실력이 없어도 괜찮고, 같이 배우고 싶은 사람
- 아이디어를 자유롭게 이야기할 수 있는 사람
